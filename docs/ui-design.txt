Fashion Carbon Calculator UI Design Specification
Project Overview
Product: Fashion Carbon Calculator MVP - A mobile-first Progressive Web App (PWA) that calculates the carbon footprint of fashion items based on material composition and manufacturing location.
Target Users: Environmentally conscious consumers who want to understand the environmental impact of their clothing purchases.
Design Pattern: Accordion/Expandable Sections with progressive disclosure
Core Design Principles
1. Progressive Disclosure
	â€¢	Show smart defaults immediately (e.g., T-shirt = 7.5 kg CO2e)
	â€¢	Reveal complexity only when users need it
	â€¢	Always provide educated defaults when users lack information
2. Mobile-First Optimization
	â€¢	Minimum 48px touch targets
	â€¢	Bottom-sheet modals for complex selections
	â€¢	Thumb-zone placement for primary actions
	â€¢	Single-column layout with vertical scroll
3. Error Prevention & Recovery
	â€¢	Real-time validation with helpful error messages
	â€¢	Auto-correction options (e.g., percentages that don't sum to 100%)
	â€¢	Clear visual feedback for invalid states
	â€¢	Maintain user inputs during error states
4. Real-time Feedback
	â€¢	Live carbon impact updates as users make selections
	â€¢	Running total in sticky header
	â€¢	Visual indicators for calculation progress
	â€¢	Smooth number animations for impact emphasis
